# מה עבר מהאתר הציבורי

הועברו או נבנו מחדש בפרויקט הנפרד:

- פורמט תאריך עברי כמו `ד׳ באב תשפ״ו`.
- תאריך לועזי ותאריך עברי יחד.
- חסימת תאריכים שעברו.
- נתוני מתחמים ראשוניים.
- בדיקת זמינות.
- סטטוסים תפעוליים.
- פניות לקוחות.
- משימות.
- PWA להתקנה במסך הבית.

מה לא הועבר בכוונה:

- עמודי שיווק.
- גלריות ותמונות.
- מודלים ציבוריים של פרטי מתחם.
- אזורי תוכן שמיועדים ללקוח באתר.

הסיבה: האפליקציה הזאת היא כלי עבודה פנימי, לא אתר שיווקי.
