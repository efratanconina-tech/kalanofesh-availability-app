# ניהול זמינות קלנופש

אפליקציית PWA פרטית לניהול זמינות מתחמי נופש.

הפרויקט הזה נפרד מהאתר הציבורי. הוא מיועד לשימוש שלכם בפלאפון: את, בעלך ובהמשך עובדים.

## מה כבר בפנים

- דשבורד ראשי.
- בדיקת זמינות מהירה ללקוח בטלפון.
- לוח זמינות לפי מתחם.
- פניות לקוחות.
- משימות.
- תאריך לועזי ותאריך עברי.
- חסימת תאריכים שעברו.
- סימון סטטוסים: תפוס, בהמתנה, הוצע, לבדיקה, תחזוקה.
- התקנה כ-PWA למסך הבית.
- נתוני מתחמים ראשוניים שהועברו מהאתר.

## הרצה מקומית

```bash
npm run dev
```

ברירת מחדל:

```text
http://127.0.0.1:5174/
```

## בנייה

```bash
npm run build
```

## מצב נתונים כרגע

הנתונים מחוברים ל-Supabase.

נבדק:

- משתמש ראשון התחבר.
- משתמש שני התחבר.
- סימון זמינות נשמר בענן.
- הסימון הופיע גם אצל המשתמש השני.

אם אין חיבור או אין משתמש מחובר, האפליקציה עדיין יכולה לעבוד מקומית לצורך בדיקות פיתוח.

## התקנה בפלאפון

כדי להתקין בפלאפון צריך קודם להעלות את האפליקציה לכתובת HTTPS, למשל דרך Vercel.

אחרי העלאה לאינטרנט:

באייפון:

1. לפתוח את הקישור ב-Safari.
2. ללחוץ Share.
3. לבחור Add to Home Screen.

באנדרואיד:

1. לפתוח את הקישור ב-Chrome.
2. ללחוץ על שלוש נקודות.
3. לבחור Install app או Add to Home Screen.
